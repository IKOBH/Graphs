add_edge(*key, overwrite_nodes=True, nodes_attributes=None, multi=False, overwrite=True, **attributes)[source]¶

Add edge with ‘key’.

Set edge’s nodes attributes using ‘nodes_attributes’. If ‘overwrite_nodes’ is True, existing nodes attributes will be overwritten. Set edge’s attributes using ‘attributes’. If ‘overwrite’ is True, edge’s attributes will be overwritten. If ‘multi’ is True, allow parallel edges.

Parameters:
  • key (Iterable[Hashable]) – edge’s key.
  • overwrite_nodes (bool) – enable edge’s nodes attributes overwriting.
  • nodes_attributes (Mapping | None.) – edge’s _nodes attributes.
  • multi (bool.) – enable parallel edge.
  • overwrite (bool) – enable edge’s attributes overwriting.
  • attributes (Mapping.) – edge’s attributes.

edge.
